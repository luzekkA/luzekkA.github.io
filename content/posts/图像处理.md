# 图像处理

最邻近缩放算法

// 简单 RGB565 最近邻缩放：将 src (sw x sh) 缩放到 dst (dw x dh)

```c
static void rgb565_scale_nearest(const uint16_t *src, int sw, int sh,
                                 uint16_t *dst, int dw, int dh)
{
    for (int y = 0; y < dh; ++y)
    {
        int sy = (y * sh) / dh; // 对应源行
        const uint16_t *src_line = src + sy * sw;
        uint16_t *dst_line = dst + y * dw;
        for (int x = 0; x < dw; ++x)
        {
            int sx = (x * sw) / dw;
            dst_line[x] = src_line[sx];
        }
    }
}
```

就是按行优先，一行一行处理，然后先计算出对应源的行与目的行

然后在行里面遍历，按照缩放取对应的X
